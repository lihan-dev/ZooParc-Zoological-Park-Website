# Zooparc Zoological Park Website 🐼
The web-based system developed for Zooparc Zoological Park to manage and display information about animals, events, and visitor services.

## 📌 Features
- Home page with attractive and user-friendly design
- Animal information pages
- Event scheduling and management
- Volunteer registration system
- Food outlet information
- Responsive layout for different devices

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL (phpMyAdmin)
- WAMP Server

## 🎯 Project Description
Zooparc is a zoological park housing over 2,000 animals from 200 different species, including giant pandas. This system was developed to enhance user engagement and provide essential information to visitors, while also supporting backend functionality using PHP and database integration.

## ▶️ How to Run
1. Clone or download the repository
2. Move the project folder to `www` directory in WAMP (e.g., `C:\wamp64\www`)
3. Start WAMP Server (Apache & MySQL)
4. Import the database using phpMyAdmin (if SQL file is provided)
5. Open browser and go to:
http://localhost/your-project-folder
